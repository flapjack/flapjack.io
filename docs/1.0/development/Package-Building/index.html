Package Building

The omnibus-flapjack repo is the home of flapjack package building. It makes use of Omnibus, which you can optionally run within a Docker container.

Consult the omnibus-flapjack Readme for details on how to build and release packages.

For some pointers on setting up a Docker environment locally see Omnibus in your Docker.

Flapjack was conceptualised by Lindsay Holmwood. Jesse Reynolds and Ali Graham are the primary developers. View the project on GitHub.
